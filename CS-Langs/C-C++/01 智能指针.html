智能指针 ​

智能指针用于增强原始指针的安全性而设计，但增强安全性意味着便捷性有所下降。

指针在生命周期结束时，不能再访问，但普通指针在其生命周期结束时不会释放内存空间，智能指针会释放。

std::unique_ptr ​

语法：

c++
std::unique_ptr<类型> 名称{初始化};

不经过初始化的指针将指向内存为0的地址空间。

初始化单个元素：std::unique_ptr<int> ptr{ new int{5} };，该区域初始化为5。

初始化数组：std::unique_ptr<int[]> ptr{ new int(5) };，声明5个大小的int型区域。

特性： ​

唯一性：不存在多个智能指针指向同一片内存区域，不同的智能指针不能相互赋值；已经声明的智能指针指向的区域不能更改。

安全性：初始化单个元素的智能指针，只能使用*访问，初始化数组类型的指针，只能用ptr[xx]访问。

std::make_unique ​

语法：

c++
std::unique_ptr<类型> 名称{ std::make_unique<类型>(大小 or 初值) };

同std::unique_ptr，初始化：

c++
std::unique_ptr<int>ptr2{ std::make_unique<int>(10) };  // 初始化元素
std::unique_ptr<int[]>ptr3{ std::make_unique<int[]>(10) };  // 初始化数组

初始化元素，代表该区域初始化为10；初始化数组，代表开辟10个int型大小的区域。

特性： ​

其可以保证声明区域的唯一性。

ptr.reset() ​

语法：

c++
ptr.reset();

将该智能指针内存区域释放，并重置该指针使其指向0地址空间。

ptr.get() ​

语法：

c++
int * ptr2;
ptr2 = ptr.get();

有时智能指针操作不方便，需要将其转化为普通指针，将智能指针指向的地址赋给普通指针

该操作不影响原本的智能指针

ptr.release() ​

语法：

c++
int * ptr2;
ptr2 = ptr.release();

该操作在ptr.get()上更进一步，不仅将智能指针指向的地址赋给普通指针，还重置了原本的智能指针使其指向0地址空间。

注意：

该方法将原本的智能指针置为0，但不会释放其指向的内存空间。

std::move(ptr) ​

语法：

c++
std::unique_ptr<int>ptr1{ std::make_unique<int>(100) };
std::unique_ptr<int>ptr2{ std::make_unique<int>(100) };
ptr1 = std::move(ptr2);

我们真的需要更换智能指针指向的区域，使用std::move()，实现赋值功能，并将原来的智能指针重置为0地址空间。

同ptr.release()，move后的原本的智能指针指向的内存区域不会释放。

代码： ​

c++
#include <iostream>

int main()
{
	std::unique_ptr<int> ptr{};

	std::cout << ptr << std::endl;

	std::unique_ptr<int> ptr2{ new int{10} };
	std::cout << ptr2 << std::endl;

	std::unique_ptr<int[]> ptr3{ new int[10] };
	std::cout << ptr3 << std::endl;

	std::cout << *ptr2 << std::endl;
	std::cout << ptr3[0] << std::endl;

	std::unique_ptr<int> ptr4{ std::make_unique<int>(100) };
	std::cout << ptr4 << std::endl;
	std::unique_ptr<int[]> ptr5{ std::make_unique<int[]>(100) };
	std::cout << ptr5 << std::endl;

	int* ptr0 = new int{ 100 };
	delete(ptr0);

	std::unique_ptr<int >ptr6{ std::make_unique<int>(500) };
	std::cout << ptr6 << std::endl;
	ptr6.reset();
	std::cout << ptr6 << std::endl;

	int a = 1000;
	std::unique_ptr<int>ptr7{ std::make_unique<int>(0) };
	int* ptr8 = &a;
	std::cout << ptr8 << std::endl;
	ptr8 = ptr7.get();
	std::cout << ptr7 << std::endl;
	std::cout << ptr8 << std::endl;
	ptr8[0] = 100;
	std::cout << *ptr7 << std::endl;

	int b = 1000;
	std::unique_ptr<int>ptr9{ std::make_unique<int>(0) };
	int* ptr10 = &b;
	std::cout << ptr9 << std::endl;
	ptr10 = ptr9.release();
	std::cout << ptr9 << std::endl;
	std::cout << ptr10 << std::endl;


	std::unique_ptr<int[]>ptr11{std::make_unique<int[]>(1)};
	std::unique_ptr<int[]>ptr12{std::make_unique<int[]>(1)};
	std::cout << ptr11 << std::endl;
	std::cout << ptr12 << std::endl;
	ptr12 = std::move(ptr11);
	std::cout << ptr11 << std::endl;
	std::cout << ptr12 << std::endl;

}
