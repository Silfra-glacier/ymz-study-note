联合体 ​

联合体和结构体类似，但联合体中的成员变量共享内存空间。

c++
union USER
{
	short a;
	int b;
	double c;
};

变量a、b、c是共享内存的，其中USER的大小为最大成员变量的大小，8B。

匿名联合体 ​

匿名联合体即没有名字的联合体，也可以声明匿名结构体。

c++
union  // 可以声明匿名联合体（结构体也可以），匿名联合体只能声明 1 次
{
	int a;
}anonymous_union;
// anonymous_union理解为使用该联合体的代号
