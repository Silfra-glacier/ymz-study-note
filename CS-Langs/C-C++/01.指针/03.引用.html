引用 ​

引用的更多实践，将在后续的函数中讲解。

引用是对一个变量起了一个“别名”，可以通过这个别名对该变量进行操作。

例子：

c++
int a{ 1200 };
int& a1 = { a };
int& a2 = { a };

std::cout << "&a:\t" << &a << std::endl;
std::cout << "&a1:\t" << &a1 << std::endl;
std::cout << "&a2:\t" << &a2 << std::endl;

输出：

&a:     00000099251EF704
&a1:    00000099251EF704
&a2:    00000099251EF704

引用所占用的内存地址和被引用的变量占用的内存地址相同。

不建议移动变量的引用到另一个变量，因为这本身无法移动，且容易引起理解混淆：

c++
int b{ 2400 };
a1 = b;  // 并不是将 a 的引用 a1 变成了 b 的引用，而是直接将 b 的值赋给了 a1 所引用的变量 a
std::cout << "a:\t" << a << std::endl;
std::cout << "b:\t" << b << std::endl;

输出：

a:      2400
b:      2400

常量引用 ​

常量引用是指对常量的引用，用法如下：

c++
const int C{ 5000 };
const int& C1{ C };
std::cout << "C:\t" << C << std::endl;
// C++;  // 常量引用不能修改
std::cout << "C1:\t" << C1 << std::endl;

输出：

C:      5000
C1:     5000

引用是否会占用内存空间？ ​

从汇编代码中看出：

assembly
    int a_ = 5;
00007FF7734C6BA6  mov         dword ptr [a_],5  # 将 5 放入 a_ 指向的内存地址
    a_ = 6;
00007FF7734C6BB0  mov         dword ptr [a_],6  # 将 6 放入 a_ 指向的内存地址
    int& b_ = a_;
00007FF7734C6BBA  lea         rax,[a_]  # 将 a_ 本身的值（而不是取将 a_ 作为内存地址的内存的值）放入 rax ，也就是将 a_ 的内存地址放入了 rax
00007FF7734C6BC1  mov         qword ptr [b_],rax  # 将 rax 的值放入 b_ 指向的内存地址，也就是将 a_ 的内存地址放入了 b_ 所在的内存空间，可以看出 b_ 变量本身也是占用了一块内存空间的，内存地址为 b_ ，[b_] 为地址中的具体值，也就是 a_
    int& c_ = a_;

00007FF7734C6BCF  mov         qword ptr [c_],rax

00007FF7734C6BD6  lea         rax,[a_]  
00007FF7734C6BDD  mov         qword ptr [d_],rax  
    b_ = 6;
00007FF7734C6BE4  mov         rax,qword ptr [b_]  # 将 b_ 地址指向的内存中的值放入 rax ，也就是将 a_ 的地址 a_ 放入 rax
00007FF7734C6BEB  mov         dword ptr [rax],6  # 将 6 放入 rax 指向的内存地址中的值，也就是将 6 放入 a_ 地址指向的空间中，即修改了 a_ 变量的值，下同
    c_ = 6;
00007FF7734C6BF1  mov         rax,qword ptr [c_]  
00007FF7734C6BF8  mov         dword ptr [rax],6  
    *d_ = 6;
00007FF7734C6BFE  mov         rax,qword ptr [d_]  
00007FF7734C6C05  mov         dword ptr [rax],6  
    a_ = (int)&a_;
00007FF7734C6C0B  lea         rax,[a_]  
00007FF7734C6C12  mov         dword ptr [a_],eax

似乎引用占用了新的内存空间，但事实上引用只是变量的别名，不会占用新的内存空间。

本节全部代码 ​

c++
#include <iostream>

int main()
{
    int a{ 1200 };
    int& a1 = { a };
    int& a2 = { a };

    std::cout << "输出每个变量所在的内存地址:" << std::endl;
    std::cout << "&a:\t" << &a << std::endl;
    std::cout << "&a1:\t" << &a1 << std::endl;
    std::cout << "&a2:\t" << &a2 << std::endl;

    a1++;
    std::cout << "a:\t" << a << std::endl;

    std::cout << "***** 引用不可移动 *****" << std::endl;
    int b{ 2400 };
    a1 = b;  // 并不是将 a 的引用 a1 变成了 b 的引用，而是直接将 b 的值赋给了 a1 所引用的变量 a
    std::cout << "a:\t" << a << std::endl;
    std::cout << "b:\t" << b << std::endl;

    std::cout << "***** 常量引用 *****" << std::endl;
    const int C{ 5000 };
    const int& C1{ C };
    std::cout << "C:\t" << C << std::endl;
    // C++;  // 常量引用不能修改
    std::cout << "C1:\t" << C1 << std::endl;

    // 引用也占用内存地址
    std::cout << "***** 引用是否会占用内存地址？ *****" << std::endl;
    int a_ = 5;
    a_ = 6;
    int& b_ = a_;
    int& c_ = a_;
    int* d_ = &a_;
    b_ = 6;
    c_ = 6;
    *d_ = 6;
    a_ = (int)&a_; 

    std::cout << &a_ << std::endl;
    std::cout << &b_ << std::endl;
}
