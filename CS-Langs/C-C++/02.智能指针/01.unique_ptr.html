01.unique_ptr

智能指针

On this page

unique_ptr ​

声明 ​

c++
std::unique_ptr<int> ptrA{ std::make_unique<int>(150) };  // 声明单个数字的指针，初始化数据为 150
std::unique_ptr<int[]> ptr_arrA{ std::make_unique<int[]>(10) };  // 声明指向数组的指针，声明的大小为 10 个 int 型

声明的指针可以像普通指针那样去访问：

c++
std::cout << *ptrA << std::endl;
ptr_arrA[1] = 1024;

注意，声明单个数字的指针只能使用 * 去访问，而不能使用 ptrA[0] 下标式的访问；同样的，声明数组指针也只能用下标式的访问，而不能用 * 访问。

唯一性 ​

c++
std::unique_ptr<int> a{ std::make_unique<int>(1) };
std::unique_ptr<int> b{};
//b = a; // ❌ 错误，不能赋值

唯一指针的唯一性体现在其所有权只能同时被一个变量持有，不可以将一个唯一指针赋值给另外一个唯一指针，使得同时存在两个唯一指针指向同一片内存区域。

reset() ​

reset() 方法用于将唯一指针释放，并回收当前内存区域：

c++
std::unique_ptr<int> c{ std::make_unique<int>(2) };
std::cout << "c:\t" << c << std::endl;
c.reset();  // 将 c 设置为 nullptr
std::cout << "c:\t" << c << std::endl;

输出：

c:      0000025BD92473A0
c:      0000000000000000

get() ​

get() 用于获取唯一指针指向的地址：

c++
std::cout << "get() 用法" << std::endl;
std::unique_ptr<int> d{ std::make_unique<int>(3) };
std::cout << "d:\t" << d.get() << std::endl;
int* p = d.get();  // 这里 p 只是获取了唯一指针 d 所在的内存地址，可以使用 p 按照普通指针的方法来操作内存，但不获得 d 的所有权，一旦 d 释放，p 也就不可使用
// 不能释放 p 所在的内存，因为 p 不拥有这片内存的所有权。当释放掉 d 的内存后，p 自然就被释放了
p[0] = 100;
std::cout << *d << std::endl;
*d = 150;
std::cout << *d << std::endl;

注意，上述代码中，虽然可以通过 get() 方法，将唯一指针 d 指向的内存地址赋值给普通指针 p ，但 p 并不会拥有 d 的所有权，也没有管理这片内存区域的权限，只能使用普通指针的方法进行读写这片内存区域，但不能释放，一旦 d 释放，则 p 也就不可再使用。

输出：

d:      0000023353557B20
100
150

realse() ​

realse() 用于将唯一指针置为 nullptr ，但不释放当前的内存区域，并返回当前唯一指针指向的这片内存区域的地址，类似于 Python 中列表的 pop() 方法。用法：

c++
std::unique_ptr<int> e{ std::make_unique<int>(4) };
std::cout << "e:\t" << e << std::endl;
int* f = e.release();
std::cout << "f:\t" << f << std::endl;
std::cout << "e:\t" << e << std::endl;
*f = 5;  // 此时可以通过指针 f 访问没有被释放的这片内存区域
std::cout << *f << std::endl;

输出：

e:      000001A450057BE0
f:      000001A450057BE0
e:      0000000000000000
5

move() ​

move() 方法用于转移唯一指针的所有权到另一个唯一指针，保持了唯一指针的唯一性。将指针的所有权转移后，指针指向的值变为 nullptr ：

c++
std::unique_ptr<int> g{ std::make_unique<int>(5) };
std::cout << "g:\t" << g << std::endl;
std::unique_ptr<int> h{ std::move(g) };  // 将 g 的所有权转移给 h
std::cout << "g:\t" << g << std::endl;  // g 置为 nullptr
std::cout << "h:\t" << h << std::endl;
*h = 100;
std::cout << *h << std::endl;

输出：

g:      000002F4ACA07960
g:      0000000000000000
h:      000002F4ACA07960
100

本节全部代码 ​

c++
#include <iostream>

int main()
{
	// 声明 unique_ptr
	std::unique_ptr<int> ptrA{ std::make_unique<int>(150) };
	std::cout << "ptrA:\t" << ptrA << std::endl;
	std::cout << *ptrA << std::endl;
	*ptrA = 200;
	std::cout << *ptrA << std::endl;

	std::cout << "声明指向数组的指针" << std::endl;
	std::unique_ptr<int[]> ptr_arrA{ std::make_unique<int[]>(10) };

	ptr_arrA[1] = 1024;
	for (int i = 0; i < 10; i++)
	{
		std::cout << ptr_arrA[i] << std::endl;
	}

	// 唯一性
	std::unique_ptr<int> a{ std::make_unique<int>(1) };
	std::unique_ptr<int> b{};
	//b = a; // ❌ 错误，不能赋值

	// reset()
	std::cout << "reset() 用法" << std::endl;
	std::unique_ptr<int> c{ std::make_unique<int>(2) };
	std::cout << "c:\t" << c << std::endl;
	c.reset();  // 将 c 设置为 nullptr
	std::cout << "c:\t" << c << std::endl;

	// get()
	std::cout << "get() 用法" << std::endl;
	std::unique_ptr<int> d{ std::make_unique<int>(3) };
	std::cout << "d:\t" << d.get() << std::endl;
	int* p = d.get();  // 这里 p 只是获取了唯一指针 d 所在的内存地址，可以使用 p 按照普通指针的方法来操作内存，但不获得 d 的所有权，一旦 d 释放，p 也就不可使用
	// 不能释放 p 所在的内存，因为 p 不拥有这片内存的所有权。当释放掉 d 的内存后，p 自然就被释放了
	p[0] = 100;
	std::cout << *d << std::endl;
	*d = 150;
	std::cout << *d << std::endl;

	// realse()
	std::cout << "release() 用法" << std::endl;
	std::unique_ptr<int> e{ std::make_unique<int>(4) };
	std::cout << "e:\t" << e << std::endl;
	int* f = e.release();
	std::cout << "f:\t" << f << std::endl;
	std::cout << "e:\t" << e << std::endl;
	*f = 5;  // 此时可以通过指针 f 访问没有被释放的这片内存区域
	std::cout << *f << std::endl;

	// move()
	std::cout << "move() 用法" << std::endl;
	std::unique_ptr<int> g{ std::make_unique<int>(5) };
	std::cout << "g:\t" << g << std::endl;
	std::unique_ptr<int> h{ std::move(g) };  // 将 g 的所有权转移给 h
	std::cout << "g:\t" << g << std::endl;  // g 置为 nullptr
	std::cout << "h:\t" << h << std::endl;
	*h = 100;
	std::cout << *h << std::endl;
}
