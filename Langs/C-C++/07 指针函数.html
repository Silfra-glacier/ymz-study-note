指针函数 ​

指针函数用于解决当传入函数的参数的占用内存过大，且要求函数功能内部不会修改传入参数的值时，对内存开销优化的问题。

问题引入： ​

函数传参原理是将传入的参数（已经分配好变量的）重新复制到函数定义的参数部分，然后函数将使用该部分值完成函数体内的运算最后传出。

c++
#include <iostream>


struct CoffeePrice
{
	int Nescafe;
	int	Starbucks;
};

// 该函数只计算结构体内的和而不改变结构体内的值
int CalcSumPrice(CoffeePrice coffeeprice)  // 计算给出结构体内所有咖啡价格总和
{
	return coffeeprice.Nescafe + coffeeprice.Starbucks;
}

int main()
{
	CoffeePrice coffee
	{
		10,	20
	};
	int result = CalcSumPrice(coffee);

	std::cout << "result=" << result << "\tNescafe=" << coffee.Nescafe << "\tStarbucks=" << coffee.Starbucks << std::endl;

}

如果结构体内的元素增加，每次调用函数将会复制更多的内存空间，造成内存开销过大。

解决方案是使用指针参数，直接使用已经分配变量的内存空间计算，返回结果，而不需要再次将原本值再复制一遍。

c++
#include <iostream>


struct CoffeePrice
{
	int Nescafe;
	int	Starbucks;
	int	Threeand;
	int	ucc;
	int	Maxwell;
	int	Yongpu;
	int	Yili;
	int	Nespresso;
	int	Yutian;
};


int CalcSumPrice(CoffeePrice* coffeeprice)  // 计算给出结构体内所有咖啡价格总和
{
	return coffeeprice->Nescafe + coffeeprice->Starbucks + coffeeprice->Maxwell + coffeeprice->Nespresso + coffeeprice->Threeand + coffeeprice->ucc + coffeeprice->Yili + coffeeprice->Yongpu + coffeeprice->Yutian;
}

int main()
{
	CoffeePrice coffee
	{
		10,	20,30,40,50,60,70,80,90
	};
	int result = CalcSumPrice(&coffee);

	std::cout << "result=" << result << "\tNescafe=" << coffee.Nescafe << "\tStarbucks=" << coffee.Starbucks << std::endl;  // 不会改变原本的值

}

第二个应用： ​

当函数内部有的参数不能将其受到影响（改变）时，使用常量指针，避免修改该值的行为。

首先，若传入指针作为参数，相当于传入原本值的地址，则在函数内部可以修改该值。

c++
#include <iostream>


struct CoffeePrice
{
	int Nescafe;
	int	Starbucks;
	int	Threeand;
	int	ucc;
	int	Maxwell;
	int	Yongpu;
	int	Yili;
	int	Nespresso;
	int	Yutian;
};

int TestCoffee(CoffeePrice* coffeeprice)
{
	coffeeprice->Maxwell = 100;
	return 1;
}

int main()
{
	CoffeePrice coffee
	{
		10,	20,30,40,50,60,70,80,90
	};
	int result = TestCoffee(&coffee);

	std::cout  << "Maxwell=" << coffee.Maxwell << std::endl;  // 将修改Maxwell的值
}

下面的例子更详细的展示了如何传入指针修改和保护值：

c++
#include <iostream>


int TestCoffee(const int* Nescafe, int* Starbucks)  // 使用const常量指针
{
	(*Starbucks) = (*Nescafe) - (*Starbucks);
    // *Nescafe = *Starbucks - *Nescafe;  // 这样就保护了Nescafe的值不会被修改
	return 1;
}

int main()
{
	int nescafe = 300;
	int starbucks = 200;
	TestCoffee(&nescafe, &starbucks);
	std::cout << "starbucks=" << starbucks << std::endl;  // starbucks的值将被修改
}
